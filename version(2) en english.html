Understanding CSS Filters

blur()

Applies a Gaussian blur. Parameter in pixels (e.g., blur(5px)). Higher values create more blur.

grayscale()

Converts to grayscale. Value in percent (0% to 100%). At 100%, image becomes completely grayscale.

invert()

Inverts colors. Value in percent (0% to 100%). At 100%, colors are completely inverted.

Combine Filters

List functions in order. Order changes result (e.g., blur() then grayscale() vs grayscale() then blur()).

CSS Filter Properties Summary

CSS filters provide a way to apply visual effects to elements. They are commonly used with images but work on any HTML element. Filters can be combined by listing multiple filter functions separated by spaces.

Performance Tip Filters can impact performance, especially on mobile devices. Use them judiciously.

Browser Support CSS filters are supported in all modern browsers, but always test across target browsers.

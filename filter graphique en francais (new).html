Comprendre les Filtres CSS

blur()

Applique un flou gaussien. Paramètre en pixels (ex : blur(5px)). Des valeurs plus élevées créent plus de flou.

grayscale()

Convertit en niveaux de gris. Valeur en pourcentage (0% à 100%). À 100%, l'image devient complètement en niveaux de gris.

invert()

Inverse les couleurs. Valeur en pourcentage (0% à 100%). À 100%, les couleurs sont complètement inversées.

Combiner les Filtres

Listez les fonctions dans l'ordre. L'ordre change le résultat (ex : blur() puis grayscale() vs grayscale() puis blur()).

Résumé des propriétés des filtres CSS

Les filtres CSS permettent d'appliquer des effets visuels aux éléments. Ils sont couramment utilisés avec les images mais fonctionnent sur n'importe quel élément HTML. Les filtres peuvent être combinés en listant plusieurs fonctions de filtre séparées par des espaces.

Astuce Performance Les filtres peuvent affecter les performances, surtout sur les appareils mobiles. Utilisez-les avec modération.

Support Navigateur Les filtres CSS sont supportés dans tous les navigateurs modernes, mais testez toujours sur les navigateurs cibles.

/* Fonctions de filtre CSS */ filter: none | [filter-function] | [filter-function-list]; /* Fonctions de filtre individuelles */ filter: blur(5px); filter: brightness(0.8); filter: contrast(200%); filter: drop-shadow(16px 16px 20px blue); filter: grayscale(50%); filter: invert(75%); filter: opacity(25%); filter: saturate(30%); /* Filtres multiples */ filter: contrast(175%) brightness(80%) blur(3px); /* Avec URL (filtres SVG) */ filter: url("filters.svg#filter-id"); /* Valeurs globales */ filter: inherit; filter: initial; filter: unset; /* Animer les filtres */ @keyframes filter-animation { 0% { filter: blur(0px) grayscale(0%); } 50% { filter: blur(5px) grayscale(50%); } 100% { filter: blur(0px) grayscale(100%); } } .element { animation: filter-animation 3s infinite; }

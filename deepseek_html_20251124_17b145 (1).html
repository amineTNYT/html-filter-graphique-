Understanding CSS Filters

blur()

Applies a Gaussian blur to the element. The parameter defines the radius of the blur.

Values: length (px, em, rem)

grayscale()

Converts the element to grayscale. 0% is original, 100% is completely grayscale.

Values: percentage (0% to 100%)

invert()

Inverts the colors in the element. 0% is original, 100% is completely inverted.

Values: percentage (0% to 100%)

Combining Filters

You can apply multiple filters by listing them in order. The order affects the final result.

Example: filter: blur(2px) grayscale(50%) invert(25%);

Complete CSS Filters Syntax & Notations

/* CSS Filter Functions */ /* Blur */ filter: blur(0px); /* No blur */ filter: blur(5px); /* 5px blur radius */ filter: blur(1rem); /* 1rem blur radius */ /* Grayscale */ filter: grayscale(0%); /* Original colors */ filter: grayscale(50%); /* 50% grayscale */ filter: grayscale(100%); /* Complete grayscale */ /* Invert */ filter: invert(0%); /* Original colors */ filter: invert(50%); /* 50% inverted */ filter: invert(100%); /* Complete inversion */ /* Brightness */ filter: brightness(50%); /* Half brightness */ filter: brightness(100%); /* Normal brightness */ filter: brightness(200%); /* Double brightness */ /* Contrast */ filter: contrast(50%); /* Half contrast */ filter: contrast(100%); /* Normal contrast */ filter: contrast(150%); /* 1.5x contrast */ /* Saturate */ filter: saturate(0%); /* Completely desaturated */ filter: saturate(100%); /* Normal saturation */ filter: saturate(200%); /* Double saturation */ /* Sepia */ filter: sepia(0%); /* No sepia */ filter: sepia(50%); /* 50% sepia */ filter: sepia(100%); /* Complete sepia */ /* Hue Rotation */ filter: hue-rotate(0deg); /* No rotation */ filter: hue-rotate(90deg); /* 90 degree rotation */ filter: hue-rotate(180deg); /* 180 degree rotation */ /* Drop Shadow */ filter: drop-shadow(2px 2px 5px rgba(0,0,0,0.5)); /* Multiple Filters */ filter: blur(2px) grayscale(50%) brightness(120%); filter: contrast(150%) saturate(200%) hue-rotate(30deg); /* No Filter */ filter: none;
